# 🤖 CHRISTOPHER'S AI PRO

**Your Personal AI Assistant - Chat, Generate Images, and Zambia Tools**

Built by Christopher | Powered by TinyLlama + HuggingFace

## ✨ Features

- **💬 Neural Chat** - Ask anything. AI replies in 5-10 seconds
- **🎨 AI Image Generator** - Create images with AI
- **📁 Memory System** - All chats and documents auto-save to your browser
- **🇿🇲 Zambia Hub** - Business ideas, Farming tips, Market prices for Zambia
- **💾 Download Chats** - Export any conversation as .txt file
- **⚡ Fast Mode** - Works even on slow 4G internet

## 🚀 How to Use

1. Open the website: [your-github-link-here]
2. Type your question in "NEURAL CHAT"
3. Wait 5-10 seconds for AI to reply
4. Go to "MY DOCUMENTS" to save and download chats

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **AI Model**: TinyLlama-1.1B-Chat-v1.0 via HuggingFace
- **Storage**: Browser LocalStorage - No account needed
- **Hosting**: GitHub Pages
